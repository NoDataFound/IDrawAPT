         |                                                                         |
         |  How To Run iDrawAPT and Build Your Hacker Art:                         |
         |                                                                         |
         |  1. Open a terminal window.                                             |
         |                                                                         |
         |  2. Clone the repo:                                                     |
         |       git clone https://github.com/NoDataFound/iDrawAPT.git             |
         |                                                                         |
         |  3. Navigate to the directory:                                          |
         |       cd iDrawAPT                                                       |
         |                                                                         |
         |  4. Run the installer:                                                  |
         |       bash installer.sh                                                 |
         |                                                                         |
         |  5. The installer will:                                                 |
         |     - Create a virtualenv (idrawaptt/)                                  |
         |     - Install dependencies from requirements.txt (Streamlit, pandas,    |
         |       pillow, colorthief, python-dotenv, openai, requests)              |
         |     - Set up gallery/ and logs/ directories                             |
         |                                                                         |
         |  6. Launch the app:                                                     |
         |       streamlit run idapt.py                                            |
         |                                                                         |
         |  7. If the browser doesn’t open, try:                                   |
         |       streamlit run idapt.py --server.address 0.0.0.0 --server.port 1337|
         |                                                                         |
         |  8. In the Streamlit UI:                                                |
         |     - Select a region and APT group (e.g., Russia, APT28).              |
         |     - Pick a style or enter a custom prompt.                            |
         |     - Upload a logo for color matching (optional).                      |
         |     - Add context (e.g., “cyber ninja with a floppy disk”).             |
         |     - Hit “Draw APT” to generate and download transparent PNGs.         |
         |                                                                         |
         |  9. Your art will be saved in:                                          |
         |     - gallery/[region]/[group]_[style]_[timestamp].png                  |
         |     - Logs in logs/                                                     |
         |                                                                         |
         |  10. Important:                                                         |
         |     - Place a favicon.png in the project root for the UI.               |
         |     - Copy .env.example to .env and add your OpenAI API key (optional). |
         |     - Keep your API key secure, never commit .env.                      |
